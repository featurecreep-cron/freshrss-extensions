# Sticky Reader

Scroll straight to articles, keep controls within reach, read distraction-free.

<!-- screenshot -->

## Features

- Smooth scroll anchoring -- opening an article scrolls it into view automatically
- Configurable scroll target: show toolbar + search bar, toolbar only, or full-screen article
- Show the current feed name in the toolbar
- Hide the redundant feed column when viewing a single feed
- Option to hide the subscription management button from the sidebar

## Installation

Copy the `xExtension-StickyReader` directory into your FreshRSS `extensions/` directory, or install it through Extension Manager. Enable it in Settings > Extensions.

## Configuration

- **Smooth scroll anchoring** -- auto-scroll to the opened article (on by default)
- **Scroll target** -- how much of the UI stays visible: search bar + toolbar, toolbar only, or full screen
- **Show feed name in toolbar** -- displays the current feed name in the control bar
- **Hide feed column** -- removes the feed name column when viewing a single feed
- **Hide subscription management** -- removes the subscription management button from the sidebar

## Compatibility

Requires FreshRSS 1.20+.
