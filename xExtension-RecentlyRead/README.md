# Recently Read

Adds a "Recently Read" view to quickly find articles you've already opened.

<!-- screenshot -->

## Features

- Adds a "Recently Read" entry to the sidebar, below Favourites
- Shows read articles sorted by when you last interacted with them
- Per-feed recently-read filtering via sidebar context menus (with Right-Click Actions)
- Highlights the sidebar entry when the recently-read view is active

## Installation

Copy the `xExtension-RecentlyRead` directory into your FreshRSS `extensions/` directory, or install it through Extension Manager. Enable it in Settings > Extensions.

## Compatibility

Requires FreshRSS 1.20+.
