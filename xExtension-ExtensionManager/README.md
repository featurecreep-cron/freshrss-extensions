# Extension Manager

Install, update, and remove FreshRSS extensions from the settings page.

<!-- screenshot -->

## Features

- Browse extensions from configured GitHub repositories
- Install extensions with one click
- Update installed extensions to the latest version
- Remove extensions you no longer need
- Supports both single-extension and multi-extension repositories
- Atomic installs with automatic rollback on failure

## Installation

Copy the `xExtension-ExtensionManager` directory into your FreshRSS `extensions/` directory, then enable it in Settings > Extensions.

## Configuration

Add GitHub repository URLs (one per line) as extension sources. Extensions from those repos will appear on the Extensions page for installation.

## Compatibility

Requires FreshRSS 1.20+.
